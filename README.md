\<ol\>
===============
OpenLayers web component.

Installation
------------

    $ yarn add @ggcity/ol

Usage
-----

We're using Parcel to bundle up this web component

```html
<ol-map
  latitude="33.778724"
  longitude="-117.960058"
  zoom="13"
  min-zoom="11"
  max-zoom="19"
  attribution-prefix="City of Garden Grove"
></ol-map>
```

Test Run
--------

    $ npm start

Go to http://localhost:1234 on your browswer.
